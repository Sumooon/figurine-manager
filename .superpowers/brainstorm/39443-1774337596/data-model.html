核心实体与字段定义

实体关系

Batch 批次

  • id
  • name 批次名称
  • imageRange 图片范围
  • totalShipping 运费
  • totalTax 税费
  • createdAt 创建时间
1 ─ 包含 ─ N

Figurine 手办

  • id
  • imageFile 图片文件名
  • imageIndex 图片内序号
  • name 名称
  • series 系列/IP
  • tags 标签[]
  • purchasePrice 买入价
  • shippingShare 运费分摊
  • taxShare 税费分摊
  • totalCost 总成本
  • status 状态
1 ─ 卖出 ─ 0..1

Trade 交易

  • id
  • figurineId 手办ID
  • sellPrice 卖出价
  • profit 利润
  • profitRate 利润率
  • buyerName 买家名
  • buyerContact 联系方式
  • soldAt 卖出时间

关键计算逻辑

// 费用分摊（支持自定义比例）

shippingShare = batch.totalShipping × (比例 / 总比例)


// 总成本

totalCost = purchasePrice + shippingShare + taxShare


// 利润 & 利润率

profit = sellPrice - totalCost

profitRate = profit / totalCost × 100%
