// 咸鱼手续费说明

咸鱼手续费 = 成交价 × 0.6% (最低0.1元)


// 实际利润计算（更新版）

总成本 = 买入价 + 运费分摊 + 税费分摊

实际收入 = 卖出价 - 咸鱼手续费

实际利润 = 实际收入 - 总成本

实际利润率 = 实际利润 / 总成本 × 100%
