技术依赖

核心框架

  • Vue 3.4+
  • TypeScript 5.0+
  • Vite 5.0+
  • Pinia
  • Vue Router

UI & 工具

  • Element Plus
  • IDB (IndexedDB 封装)
  • ExcelJS (导出)
  • dayjs (日期)

浏览器 API

  • File System Access API
  • IndexedDB
  • LocalStorage (配置)
