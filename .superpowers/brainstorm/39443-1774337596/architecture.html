手办管理系统 - 整体设计

Vue 3 + Vite + IndexedDB 纯前端方案

系统架构

数据层

  • IndexedDB - 手办/批次/交易数据
  • File System API - 图片文件夹引用

状态层

  • Pinia Store - 全局状态管理
  • Computed - 实时计算利润/统计

视图层

  • Vue 3 + Element Plus
  • 响应式布局

核心模块

1

手办管理

手办信息录入、编辑、图片关联、状态跟踪

2

批次管理

批次创建、费用分摊、批量操作

3

统计看板

投入/收益/利润汇总、图表展示

4

交易记录

买入/卖出记录、买家信息管理

数据流示意

选择图片文件夹
→
解析图片名生成手办
→
填写详细信息
→
保存到 IndexedDB
