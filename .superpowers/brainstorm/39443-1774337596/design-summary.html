设计总结

手办管理系统 - 完整设计方案

方案概览

技术方案

  • Vue 3 + TypeScript + Vite
  • Element Plus UI 组件库
  • IndexedDB 本地存储
  • File System Access API 访问图片
  • 纯前端，打开即用

核心功能

  • 手办管理（录入、编辑、状态跟踪）
  • 批次管理（费用分摊、批量操作）
  • 统计看板（投入/收益/利润）
  • 交易记录（买家信息、咸鱼集成）
  • 导出报表（Excel/CSV/JSON）

数据模型

Batch：id, name, imageRange, totalShipping, totalTax, createdAt
Figurine：id, imageFile, imageIndex, name, series, tags, purchasePrice, shippingShare, taxShare, totalCost, status
Trade：id, figurineId, sellPrice, xianyuLink, xianyuOrderId, xianyuFee, actualIncome, profit, buyerName, buyerContact, soldAt
